# Monitoreo IUSASOL — Carpatin Adaptado

Este repositorio contiene una versión adaptada del dashboard para las necesidades de IUSASOL.

## Demo

- Dashboard: https://reportes.iusasol.mx/

## Quick start

- Clone project: `git clone <tu-repositorio-o-ruta>`
- Instala dependencias: `npm install`
- Ejecuta en desarrollo: `npm run dev`
- Abre: `http://localhost:3000`
